Getting Started

Install

Vital UI Kit 提供了兩種使用方式，您可以依照喜好或是情況選擇適合您的方式開始應用。

  • 1. 下載 Vital UI Kit

    如果您不需要開發原始碼或是還不熟悉Sass，您可以使用此種方式，方便且快速就可以應用 Vital UI Kit 在您的產品或專案上。

    下載
  • 2. 安裝 Gulp

    • Step 1. 在使用 Gulp 之前，請先至 Vital UI Kit ，利用版本控管工具 clone 整份檔案到您的本地端(建議使用Https)。

    • Step 2. 下載後，請打開您的終端機並且將路徑改成 Step 1.的資料夾，並輸入：

      npm install

      這個動作我們會幫您建立 Vital UI Kit 所需要用到的工具：Node.js 與 Bower

      您可以至Gulp, Node.js, NPM 以及 Bower進一步了解它們。

    • Step 3. 接著再執行

      npm start

      專案便會開始啟動！

  • 在您的 Html 中引入
    <link rel="stylesheet" href="./css/basic.css">
    <script src="./js/jquery.min.js"></script>
    <script src="./js/vital-ui-kit.js"></script>

    接下來只要按照文件中的寫法，即可立刻看到套用 Vital UI Kit 的效果了！

Contents

  • Compressed Folder

    如果您是經由Install的第一種方式：下載 Vial UI Kit，那麼您解壓縮後會獲得以下內容：

    
vital-ui-kit/
├── css/
│   ├── basic.min.css
│   └── basic.min.css.map
└── js/
│   └── vital-ui-kit.min.js
└── fonts/
		├── icomoon.eot
		├── icomoon.svg
		├── icomoon.ttf
		└── icomoon.woff
  • Source Code Folder

    如果您是利用 Gulp 方式，則以下會是您所看見得資料夾結構：

    
vital-ui-kit/
├── jade/
├── js/
├── sass/
└── static/

    原始碼中，我們使用了 Jade(Pug)這個 Template Engine 來產出 Html，sass 則採取 scss 寫法並經編譯產出 css。static 則放置其他外部來源或不太變動的元素，例如 fonts 與 images。

  • Sass Folder

    Vital UI Kit 最精華的部分，開發者若想變動 theme 或是風格，此處得特別暸解：

    
sass/
├── base/
├── module/
└── plugin/
		├── _vital-ui-kit.custom.scss
		├── _vital-ui-kit.variables.scss
		└── vital-ui-kit/
				├── base/
				├── components/
				├── pages/
				├── scripts/
				├── structures/
				└── utils
